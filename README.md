# CareerConnect – Full Stack Recruitment Portal

## Overview

CareerConnect is a Full Stack Recruitment Portal that connects candidates and recruiters through a modern web-based platform. The application allows recruiters to post jobs, manage applicants, and track hiring processes, while candidates can search jobs, apply for opportunities, and monitor application status.

## Features

### Candidate Module

* Candidate Registration & Login
* Browse Available Jobs
* Apply for Jobs
* Track Application Status
* View Profile Information

### Recruiter Module

* Recruiter Registration & Login
* Post New Jobs
* Manage Posted Jobs
* View Applicants
* Shortlist, Reject, or Hire Candidates
* Track Recruitment Activities

### Admin Module

* Manage Users
* Manage Jobs
* Monitor Platform Activity

## Installation

### Backend

1. Open backend project in IntelliJ IDEA.
2. Configure MySQL database in application.properties.
3. Run the Spring Boot application.

### Frontend

1. Open terminal inside careerconnect-frontend.
2. Install dependencies:

npm install

3. Start the application:

npm run dev
